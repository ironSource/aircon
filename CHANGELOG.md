Version 1.0.0
----------------------------

   * Hello world!
   * XML injection.
   * Supported configs:
        * @BooleanConfig
        * @ColorConfig
        * @FloatConfig
        * @IntConfig
        * @IntEnumConfig
        * @JsonConfig
        * @LongConfig
        * @StringConfig
        * @StringEnumConfig
        * @StringSetConfig
        * @TextConfig
        * @TimeConfig
        * @UrlConfig
   * @DefaultRes - for defining a resource as default value
   * @DefaultConfig - for defining another config as a default value
   * @ConfigGroup - for grouping configs in a generated POJO
